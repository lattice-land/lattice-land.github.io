template<typename VT , typename Alloc >
CUDA thrust::optional<LogicalElement> lala::ZInc< VT, Alloc >::interpret ( Approx  appx,
const Formula &  f 
)
inline

Expects a predicate of the form x <op> i where x is any variable's name, and i an integer.

  • If appx is EXACT: op can be >= or >.
  • If appx is UNDER: op can be >=, > or !=.
  • If appx is OVER: op can be >=, > or ==. An empty optional is also returned if the element would overflow in int_type.

template<typename VT , typename Alloc >
CUDA bool lala::ZInc< VT, Alloc >::is_underappx ( ) const
inline

An abstract domain can be eventually under-approximating which means that after a sufficient number of split and refine operations, it always reach an under-approximating element.

Returns
true if \(\gamma(a) \subseteq [\![\varphi]\!]^\flat\). false is returned whenever a is an over-approximation or if we do not know whether a is an under-approximation.
