◆ deinterpret() [1/2]

template<class PreUniverse , class Mem >
template<class Env >
CUDA NI TFormula< typename Env::allocator_type > lala::FlatUniverse< PreUniverse, Mem >::deinterpret ( AVar  avar,
const Env &  env 
) const
inline
Returns
\( x = k \) where x is a variable's name and k the current value. true is returned whenever \( a = \bot \), and false is returned whenever \( a = \top \). We always return an exact approximation, hence for any formula \( \llbracket \varphi \rrbracket = a \), we must have \( a = \llbracket \rrbracket a \llbracket \rrbracket \) where \( \rrbracket a \llbracket \) is the deinterpretation function.
