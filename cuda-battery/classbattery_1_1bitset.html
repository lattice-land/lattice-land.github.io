Detailed Description

template<size_t N, class Mem, class T = unsigned long long>
class battery::bitset< N, Mem, T >

Template Parameters
<tt>N</tt>is the number of bits in the bitset.
<tt>Mem</tt>is a memory consistency, for instance local_memory, see memory.hpp.
<tt>T</tt>is the underlying type defining the blocks of the bitset.
