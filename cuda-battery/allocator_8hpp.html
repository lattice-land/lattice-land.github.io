Detailed Description

We provide a bunch of allocator compatible with the structure of the lattice land project. The allocators are aimed to be used to distinguish in which memory (shared, global, managed or the "standard" C++ memory) we should allocate data. This allows us to provide uniform interfaces for both host (C++) and device (CUDA) code.
