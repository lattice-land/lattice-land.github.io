Detailed Description

An allocator allocating memory from a pool of memory. The memory can for instance be the CUDA shared memory. This allocator is rather incomplete as it never frees the memory allocated. It allocates a control block using the "normal" operator new, where the address to the pool, its capacity and current offset are stored.
